Labseq Application Tutorial

The Labseq Application is a web application that combines a Quarkus backend with an Angular frontend. The main goal of the application is to provide an interface to calculate and return values from a specific sequence called "labseq".

How It Works

The application allows users to input a number, and it returns the corresponding value from the labseq sequence. The labseq sequence is defined as follows:

  • labseq(0) = 0
  • labseq(1) = 1
  • labseq(2) = 0
  • labseq(3) = 1
  • For n > 3, labseq(n) = labseq(n-4) + labseq(n-2)

Expected Results

When you input a number, the application will calculate and display the corresponding value from the labseq sequence. For example:

  • Input: 0, Output: 0
  • Input: 1, Output: 1
  • Input: 4, Output: 1
  • Input: 5, Output: 1
  • Input: 6, Output: 0
